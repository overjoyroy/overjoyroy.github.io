## Pokémon Card Checklist

I built this because I kept forgetting which cards I was still missing and wanted a quick reference I could pull up anywhere. I only built this for my own convenience so far. For now it only shows what I'm still hunting for but I'm working on a bigger version that'll track what I already have too, and eventually open it up to other collectors.

I collect vintage Pokémon cards from the sets I grew up with when I was a kid. Unfortunately, I have no interest in selling my collection and I won't hear any offers.
